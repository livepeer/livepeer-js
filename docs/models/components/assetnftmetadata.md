# AssetNftMetadata

Additional data to add to the NFT metadata exported to
IPFS. Will be deep merged with the default metadata
exported.


## Example Usage

```typescript
import { AssetNftMetadata } from "livepeer/models/components";

let value: AssetNftMetadata = {};
```
